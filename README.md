<h1 align="center">🗂️ FileManager</h1>
<p align="center">Консольный файловый менеджер на C++</p>

---

## 🚀 Возможности

### 📁 Навигация

* `ls` / `dir` — список файлов с иконками по типу
* `cd <путь>` — переход в директорию
* `cd ..` — переход на уровень выше
* `open <N>` — открыть файл или зайти в папку

### 📄 Работа с файлами

* `info <N>` — информация (размер, права, дата)
* `preview <N>` — предпросмотр текстового файла

### 🛠️ Операции

* `cp` — копирование
* `mv` — перемещение
* `rm` — удаление
* `mkdir` — создать папку
* `rename` — переименовать

### 🔍 Поиск

* `find *.txt` — поиск по имени (wildcard `*`)
* `findc TODO cpp` — поиск текста внутри файлов

### 👁️ Дополнительно

* `hidden` — показать / скрыть скрытые файлы

## 📂 Структура проекта

```
filemanager/
├── common.h        # общие типы, ANSI-цвета, утилиты
├── logger.h/.cpp   # JSON-логирование
├── fileops.h/.cpp  # файловые операции
├── search.h/.cpp   # поиск (имя + содержимое)
├── ui.h/.cpp       # консольный интерфейс
├── main.cpp        # REPL-цикл
├── CMakeLists.txt  # сборка через CMake
├── Makefile        # сборка через make
└── README.md       # документация
```

## ⚙️ Сборка

### 🔹 Через CMake

```bash
mkdir build
cd build
cmake ..
make
```

### 🔹 Через Makefile

```bash
make
```
